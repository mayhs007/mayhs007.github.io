Farming Simulation Game

Tools Used: Unity Engine, C#, SpriteKit, Tilemaps, Animator, Unity UI

Overview: Agrosquire is a 2D farming simulation game that I built using Unity and C#. It features a core gameplay loop around planting, watering, harvesting crops, managing time cycles, and interacting with a dynamic environment.

Key Features:

  • Inventory System with dynamic slots
  • Day/Night & Season Cycle
  • Farming Logic with growth stages
  • Player Movement & Animation using Animator Controller
  • Custom Unity scripts using OOP design principles
